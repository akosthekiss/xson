xson.dump(obj, fp, *, skipkeys=False, check_circular=True, allow_nan=True, indent=None, default=None, sort_keys=False)¶

Serialize a value to a file in JSONx format.

Parameters

  • obj – Value to be serialized.

  • fp – File-like object to write JSONx str to.

  • skipkeys (bool) – If true, then dictionary keys that are not of a basic type (str, int, float, bool, None) will be skipped. Otherwise, a TypeError is raised. (Default: False)

  • check_circular (bool) – If false, then the circular reference check for container types will be skipped. Otherwise, a ValueError is raised. (Default: True)

  • allow_nan (bool) – If false, then it will be a ValueError to serialize out-of-range float values (nan, inf, -inf). Otherwise, their JavaScript equivalents (NaN, Infinity, -Infinity) will be used. (Default: True)

  • indent (int or str) – If a positive integer, then JSON array elements and object members will be pretty-printed with that many spaces per level. If a string, that string is used to indent each level. An indent level of 0, negative, or "" will only insert newlines. None selects the most compact representation. (Default: None)

  • default – If specified, it must be a function that gets called for objects that can’t otherwise be serialized, and it must return an encodable version of the object. If not specified, TypeError is raised. (Default: None)

  • sort_keys (bool) – If true, then the output of dictionaries will be sorted by key. (Default: False)

xson.load(fp, *, object_hook=None, parse_float=None, parse_int=None, parse_constant=None, object_pairs_hook=None)¶

Deserialize a JSONx file to a Python object.

Parameters

  • fp – File-like object to be deserialized.

  • object_hook – If specified, it must be a function that will be called with the result of any object decoded (a dict), and its return value will be used instead. (Default: None)

  • object_pairs_hook – If specified, it must be a function that will be called with the result of any object decoded into an ordered list of pairs, and its return value will be used instead. If object_hook is also specified, object_pairs_hook takes priority. (Default: None)

  • parse_float – If specified, it must be a function that will be called with the string of every float to be decoded. (Default: float)

  • parse_int – If specified, it must be a function that will be called with the string of every int to be decoded. (Default: int)

  • parse_constant – If specified, it must be a function that will be called with one of the following strings: '-Infinity', 'Infinity', or 'NaN'. (Default: float)

Returns

The value deserialized.

Raises

ValueError – If the data being deserialized is not a valid JSONx document.
